# mapserverproj-org
mapserver plus proj.org

This Dockerfile is the joint of the mapserver (found in the DockerHub: https://hub.docker.com/r/camptocamp/mapserver) and the proj.org files (also at: https://hub.docker.com/r/osgeo/proj/)

Volume at : ./docker/mapfile
inside container at : /map

```
  docker-compose up
```
